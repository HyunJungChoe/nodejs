## 실행 방법

1. 몽고 디비 다운
2.  C:\Program Files\MongoDB\Server\4.2\bin 에서 cmd 열어서 mongod  : 서버 실행
3. 프로젝트 파일 연 후 터미널에서 npm i  ->  npm start
